You can integrate this background removal functionality into any website using our API:

JavaScript Example:

const formData = new FormData(); formData.append('image', imageFile); formData.append('model', 'u2net'); formData.append('format', 'png'); formData.append('return_base64', 'true'); fetch('YOUR_API_URL/remove-background', { method: 'POST', body: formData }) .then(response => response.json()) .then(data => { if (data.success) { document.getElementById('result').src = data.image_data; } });

cURL Example:

curl -X POST YOUR_API_URL/remove-background \ -F "image=@your_image.jpg" \ -F "model=u2net" \ -F "format=png"
